# How to install:
1. ``` git clone https://github.com/Dominikdoms/ProjektKoncowy.git ```
2. ```  cd my-app ```
3. ``` npm i ```
4. ``` json-server db.json --watch ```
5. open two terminal: ``` npm start ```
